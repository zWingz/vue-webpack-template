# vue webpack模板


----
## 运行环境
    node>=7.6.0
    npm>=5.0

## 主要依赖

`vue@2.4.2` `webpack@3.3.0` `vue-router@2.7.0` `axios@0.16.2"`
`vuex@2.3.1

----

## Build Setup

+ git clone拉取项目

+ cd dir,执行

    `npm install`
    
    完成依赖包安装
    

+ 开发环境(访问 localhost:8080)

    `npm run dev`

+ 生产环境

    `npm run build`
