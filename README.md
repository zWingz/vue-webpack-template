# vue-webpack项目模板


### 运行环境
    node>=7.6.0
    npm>=5.0
    webpack>=3.3.0

### Build Setup

+ 安装[cli](git@github.com:zWingz/create-app-cli.git) (也可以使用vueCli)

+ 执行
    ```bash
    create-app init webpack-template <project-name> -c
    cd project-name
    npm install
    npm run dev
    ```
